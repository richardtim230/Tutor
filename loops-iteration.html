Connecting with Previous Topics

So far, you’ve learned how to use variables to store values, operators to change or compare them, functions to group actions, and events to trigger code. What if you want to do the same thing multiple times—such as print every item in a list, or keep asking the user for input until they give a valid answer? That’s where loops come in. Loops let you run code more than once, saving you from repetition and making your code smarter and shorter.

Loops & Iteration: What and Why?

A loop repeats a block of code for you. Each turn is called an iteration. Without loops, even a tiny task would take hundreds of lines. 

// Printing numbers 1 through 5, the slow way
console.log(1);
console.log(2);
console.log(3);
console.log(4);
console.log(5);

// With a loop
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
With loops, you get more done with much less code.

The for Loop

The for loop is the most common way to repeat something a set number of times. It takes three parts: 

for (let i = 0; i < 3; i++) {
  // code runs three times: i = 0, 1, 2
  console.log("Count is", i);
}
Why?
- The first part (let i = 0) sets the starting value
- The middle (i < 3) is the stopping rule
- The last (i++) updates the counter

The while Loop

Use a while loop when you don’t know exactly how many times you’ll need to repeat. 

let n = 5;
while (n > 0) {
  console.log("Counting down:", n);
  n = n - 1;
}
Why?
The loop goes on “while” the condition is true. As soon as n drops to 0, it stops.

for...of & forEach

Loops also help you go through arrays (lists). These two are especially great for that: 

let colors = ['red', 'blue', 'yellow'];
// for...of
for (let color of colors) {
  console.log("Color:", color);
}
// forEach (more concise)
colors.forEach(function(color, i) {
  console.log("Index", i, "is", color);
});
Why?
for...of gives you each value. forEach gives both value and position.

break and continue

Sometimes you want to “jump out of” or “skip” in the middle of a loop. Use break to exit, continue to skip to the next turn. 

for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // skips 3
  if (i === 5) break;    // stops entirely at 5
  console.log(i);
}
// Output: 1, 2, 4
Why?
break lets you leave early. continue lets you skip without stopping the whole loop.

<!DOCTYPE html> <html lang="en"> <body> <div id="tools"></div> <script> let items = ["pen", "book", "eraser"], out = ''; items.forEach(function(thing) { out += "Item: " + thing + '<br>'; }); document.getElementById('tools').innerHTML = out; </script> </body> </html>
