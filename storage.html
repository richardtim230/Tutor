What is Browser Storage?

Local Storage and Session Storage let your JavaScript code save text inside the browser. It stays put even if the user leaves or reloads your page (for localStorage), or until the browser tab closes (for sessionStorage). This allows you to:
  • Remember login or theme settings
  • Save a shopping cart or form data in progress
  • Cache fetched API data for faster returns
  • Build offline-friendly experiences
localStorage: Keeps data after a page reload or browser restart
sessionStorage: Removes data when the tab/window closes

Pick localStorage for "remember tomorrow" data, and sessionStorage for "just remember while I'm here"!

Reading & Writing Simple Data

Both APIs use simple string methods: 

// Store data
localStorage.setItem("theme", "dark");
sessionStorage.setItem("user", "Lizzy");

// Get data
let myTheme = localStorage.getItem("theme");      // "dark"
let tempUser = sessionStorage.getItem("user");    // "Lizzy"

// Remove data
localStorage.removeItem("theme");                 // deletes "theme"
sessionStorage.clear();                           // clears all sessionStorage

// Check if a value is saved
if(localStorage.getItem("theme")) {
  // theme is saved
}
All stored values are always strings. Numbers and booleans must be converted (using .toString()/parseInt()/parseFloat()).

Saving Objects & Arrays (with JSON)

JavaScript objects and arrays can't go into browser storage directly; you have to turn them into JSON strings first, and turn them back into objects when you read them. 

// Save an array
let cart = ["rice", "beans", "bread"];
localStorage.setItem("cart", JSON.stringify(cart));

// Get the array back
let myCart = JSON.parse(localStorage.getItem("cart")); // ["rice","beans","bread"]

// Save an object
let profile = { name:"Jay", loggedIn:true };
sessionStorage.setItem("profile", JSON.stringify(profile));
// Fetch
let saved = JSON.parse(sessionStorage.getItem("profile"));

Updating, Removing & Clearing Data

To update, just set the same key again. To delete one item, use removeItem. To delete everything, call clear(). 

// Update profile
localStorage.setItem("profile", JSON.stringify({name:"Liz",score:8}));

// Remove a single item
localStorage.removeItem("profile");

// Clear all localStorage data
localStorage.clear();

Tips & Best Practices

  • Don’t put passwords, tokens or very sensitive info in browser storage (it’s not secure).
  • Always JSON.stringify() arrays/objects before saving, and JSON.parse() when reading them.
  • Each domain/site has its own storage. Data is not shared across sites.
  • Watch the size: both options are for small/intermediate data only (5-10 MB limit).
  • Use clear() to let users log out or reset easily.
  • To see stored data: open browser DevTools > Application tab.

Practice (Show Hidden Solutions)

1. Save a username "Chika" in localStorage. Read it and display it on a webpage.
<!DOCTYPE html> <html lang="en"> <body> <div id="name"></div> <script> localStorage.setItem("username", "Chika"); document.getElementById('name').textContent = localStorage.getItem("username"); </script> </body> </html>
2. Save an array of favorite colors ["green", "blue", "gold"] in sessionStorage and show as a comma-separated string.
<!DOCTYPE html> <html lang="en"> <body> <div id="favcol"></div> <script> sessionStorage.setItem("favColors", JSON.stringify(["green", "blue", "gold"])); let arr = JSON.parse(sessionStorage.getItem("favColors")); document.getElementById('favcol').textContent = arr.join(", "); </script> </body> </html>
3. Update one of the colors in the array above and display the updated array.
<!DOCTYPE html> <html lang="en"> <body> <div id="setcol"></div> <script> let arr = JSON.parse(sessionStorage.getItem("favColors")); arr[2] = "purple"; sessionStorage.setItem("favColors", JSON.stringify(arr)); document.getElementById('setcol').textContent = arr.join(", "); </script> </body> </html>
4. Remove the saved score from localStorage.
<!DOCTYPE html> <html lang="en"> <body> <script> localStorage.removeItem("score"); </script> <div>Score removed!</div> </body> </html>
Try some JSON.stringify() and JSON.parse() combos with your own data—learning storage is a perfect way to make your JavaScript projects “remember” things, just like professional apps do!
