Connecting with Previous Topics

Variables store single values, arrays are used for lists, loops are for repetition, and functions organize reusable code. Objects are the next step, letting you group different, named properties or methods together. This structure keeps complex code organized and readable.

What Are Objects?

An object organizes information with key-value pairs called properties. It can also have methods, which are actions attached to an object. 

let car = {
  brand: "Toyota",
  model: "Corolla",
  year: 2021
};
This is more efficient and more readable than declaring multiple separate variables for each property.
You can build objects for anything, such as users, posts, settings, transactions, animals, events, or products. 

let profile = {
  username: "codecxtutor",
  age: 25,
  isAdmin: false,
  tags: ["moderator", "active"]
};

let article = {
  id: 3,
  title: "Intro to JS Objects",
  published: true,
  comments: []
};

Creating and Modifying Objects

You can use curly braces to make a new object, assigning values with colons. Add properties or methods after object creation using dot or bracket notation. 

let book = {};
book.title = "Beginner JavaScript";
book.year = 2020;
book["pages"] = 150;
To update or remove properties: 

book.year = 2022;            // update
delete book.pages;           // remove
To check if a property exists: 

"title" in book;             // true
"author" in book;            // false

Reading Properties and Access Patterns

Use dot notation when you know the property name: 

let user = {name: "Aminat", age: 18};
console.log(user.name);            // "Aminat"
Use bracket notation if the key is stored in a variable or has special characters: 

let item = { "data-type": "json" };
console.log(item["data-type"]);
let prop = "age";
console.log(user[prop]);

Object Methods

A method is a function attached to an object. Methods let objects perform actions using their own data. 

let student = {
  name: "Frank",
  greet: function() {
    return "Hello, I am " + this.name;
  }
};
student.greet(); // "Hello, I am Frank"
this lets you refer to the object the method belongs to. 

let circle = {
  radius: 5,
  area: function() {
    return Math.PI * this.radius ** 2;
  }
};
circle.area(); // 78.5398...

Looping Through Object Properties

Use for...in to loop through all keys in an object. For getting property names and values as arrays, use static methods: 

let phone = { brand: "Samsung", os: "Android", price: 500 };
for (let key in phone) {
  console.log(key + ": " + phone[key]);
}
Object.keys(phone); // ["brand","os","price"]
Object.values(phone); // ["Samsung","Android",500]
Object.entries(phone); // [["brand","Samsung"],["os","Android"],["price",500]]
These array methods make it easy to transform or search all properties at once.

Practice (with Hidden Solutions)

1. Define an object pet with keys name and species. Print both values using dot notation.
<!DOCTYPE html> <html lang="en"> <body> <div id="petOut"></div> <script> let pet = { name: "Rex", species: "dog" }; document.getElementById('petOut').textContent = pet.name + " " + pet.species; </script> </body> </html>
2. Add a method describe to pet that returns a string like "Rex is a dog".
<!DOCTYPE html> <html lang="en"> <body> <div id="descOut"></div> <script> let pet = { name: "Rex", species: "dog", describe: function() { return this.name + " is a " + this.species; } }; document.getElementById('descOut').textContent = pet.describe(); </script> </body> </html>
3. Given order = {id:1, total:700, status:"paid"}, print all keys and values using a loop.
<!DOCTYPE html> <html lang="en"> <body> <div id="orderOut"></div> <script> let order = {id:1, total:700, status:"paid"}; let out = ""; for (let key in order) { out += key + ": " + order[key] + "<br>"; } document.getElementById('orderOut').innerHTML = out; </script> </body> </html>
4. Using Object.entries(), display each property and value of settings = {theme:"dark", font:"Arial"} as "theme: dark", "font: Arial".
<!DOCTYPE html> <html lang="en"> <body> <div id="setOut"></div> <script> let settings = {theme:"dark", font:"Arial"}; let output = ""; for (let [k, v] of Object.entries(settings)) { output += k + ": " + v + "<br>"; } document.getElementById('setOut').innerHTML = output; </script> </body> </html>
Practice writing and running these examples before expanding the solutions.
