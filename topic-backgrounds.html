What Are Backgrounds & Borders in CSS?

Backgrounds and Borders are essential for visually organizing content, separating areas, and adding personality to your web pages. With CSS, you can set colors, gradients, images, and multiple layers behind elements. Borders let you control outlines, shapes, and emphasis—giving your UI structure and flair. All these properties work together to create readable, engaging, and professional layouts.

  • Backgrounds can be solid colors, gradients, images, or even multiple layers.
  • Borders define edges, add separation, and can be styled in many creative ways.
  • Both are controlled entirely with CSS, supporting responsive and dynamic designs.

Let’s explore how to use backgrounds and borders for beautiful, accessible web design!

CSS Backgrounds

The background-* family of CSS properties lets you set backgrounds on any element. Here are the most common uses:

Background Color

This has a green background!
HTML: 
<div class="box">This has a green background!</div>
CSS: 

.box {
  background-color: #34d399;
  color: #fff;
}

Background Image

This uses a background image!
HTML: 
<div class="hero">This uses a background image!</div>
CSS: 

.hero {
  background-image: url('image.jpg');
  background-size: cover;
  background-position: center;
  color: #fff;
}

Background Gradient

Gradient Background Example
HTML: 
<div class="gradient">Gradient Background Example</div>
CSS: 

.gradient {
  background: linear-gradient(120deg, #34d399 0%, #6366f1 80%);
  color: #fff;
}

Multiple Backgrounds

Multiple backgrounds layered!
HTML: 
<div class="multi-bg">Multiple backgrounds layered!</div>
CSS: 

.multi-bg {
  background: url('pattern.png') left top no-repeat,
              linear-gradient(120deg, #34d399, #6366f1);
  color: #fff;
}

Tip: You can combine images, gradients, and colors for creative effects.

CSS Borders

Borders frame elements and are highly customizable. Here’s how to use the core border properties:

Basic Border

This box has a dashed, colored border!
HTML: 
<div class="card">This box has a dashed, colored border!</div>
CSS: 

.card {
  border: 4px dashed #22d3ee;
  border-radius: 16px;
  padding: 1.5rem;
  background: #f0fdfa;
  color: #0e7490;
  font-weight: bold;
  text-align: center;
}

Rounded Borders (border-radius)

Border Radius Demo
HTML: 
<div class="circle-box">Border Radius Demo</div>
CSS: 

.circle-box {
  border: 2px solid #6366f1;
  border-radius: 30px;
  background: #e0e7ff;
  color: #6366f1;
  font-weight: bold;
  text-align: center;
  padding: 1rem;
}

Border on Specific Sides

This box has only a bottom border!
HTML: 
<div class="underline">This box has only a bottom border!</div>
CSS: 

.underline {
  border-bottom: 2px solid #22d3ee;
  color: #22d3ee;
  font-weight: bold;
  text-align: center;
  padding: 1rem;
}

Border Styles: solid, dashed, dotted, double, groove, ridge, inset, outset.

Double border style!
HTML: 
<div class="fancy-border">Double border style!</div>
CSS: 

.fancy-border {
  border: 3px double #6366f1;
  color: #6366f1;
  font-weight: bold;
  text-align: center;
  padding: 1rem;
}

Advanced Background & Border Effects

CSS lets you create stunning effects—try border images, box shadows, and more!

Border Image

Border image effect!
HTML: 
<div class="image-border">Border image effect!</div>
CSS: 

.image-border {
  border: 10px solid transparent;
  border-image: url('border.png') 30 round;
  font-weight: bold;
  text-align: center;
  padding: 1rem;
}

Box Shadow

Box Shadow Effect!
HTML: 
<div class="shadowed">Box Shadow Effect!</div>
CSS: 

.shadowed {
  box-shadow: 0 6px 24px rgba(34,211,238,0.23);
  background: #e0e7ff;
  color: #2563eb;
  font-weight: bold;
  border-radius: 0.8rem;
  text-align: center;
  padding: 1.5rem;
}

Refactoring Legacy Code

Old HTML (using bgcolor and border attributes):


<table bgcolor="#34d399" border="2">
  <tr><td>Hello!</td></tr>
</table>

Modern CSS approach:


<table style="background-color: #34d399; border: 2px solid #22d3ee;">
  <tr><td>Hello!</td></tr>
</table>

Tip: Always use CSS for backgrounds and borders—never mix HTML and presentation.

Best Practices for Backgrounds & Borders

  • Keep presentation and markup separate: Use CSS for all backgrounds and borders.
  • Prefer external stylesheets for maintainable code.
  • Use semantic HTML and descriptive class names.
  • Test your backgrounds and borders on different devices and browsers.
  • Combine backgrounds and borders for visually appealing designs.
  • Explore CSS frameworks (Tailwind, Bootstrap) for ready-made styles and utilities.
