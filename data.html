Why Use Data Visualization?

Charts and graphs make it easier for users (and you!) to spot trends, compare values, or summarize data at a glance. Even simple datasets are easier to understand as visuals. Two of the most popular libraries for JavaScript are Chart.js and D3.js.
  • Chart.js: Easy, fast, out-of-the-box charts (bar, line, pie, doughnut, radar, and more)
  • D3.js: Powerful, detailed, highly customizable (draw just about anything you imagine)

Chart.js — Simple and Quick Charts

Chart.js is a library for putting data into attractive bar, line, pie, or other charts with only a few lines of code. Use it to create charts that look good—and work—on desktop and mobile. 
<canvas id="myChart" width="320" height="140"></canvas>
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
<script>
  const ctx = document.getElementById('myChart').getContext('2d');
  new Chart(ctx, {
    type: "bar",
    data: {
      labels: ["Math", "English", "Science"],
      datasets: [{
        label: 'Scores',
        data: [88, 94, 86],
        backgroundColor: ["#38bdf8","#fb7185","#34d399"]
      }]
    },
    options: { responsive: true }
  });
</script>
Common chart types: bar, line, pie, doughnut, radar, polar area. The data is just arrays—easy to hook up to APIs.
You can customize colors, labels, tooltips, and add "plugins" to Chart.js for more interactivity.

D3.js — Powerful, Custom Visuals

D3 gives you low-level control to build almost any visualization from scratch. Unlike Chart.js, you build up SVG elements (rectangles, lines, text, etc.). D3 is perfect when you want to craft something beyond basic chart styles. 
<svg id="d3demosvg" width="320" height="140"></svg>
<script src="https://d3js.org/d3.v7.min.js"></script>
<script>
const data = [88, 94, 86];
const svg = d3.select("#d3demosvg");
const x = d3.scaleBand().domain([0,1,2]).range([20,295]).padding(0.3);
const y = d3.scaleLinear().domain([0,100]).range([120,10]);

svg.selectAll(".bar")
  .data(data)
  .enter()
  .append("rect")
  .attr("class","bar")
  .attr("x",(d,i)=>x(i))
  .attr("y",d=>y(d))
  .attr("width",x.bandwidth())
  .attr("height",d=>120-y(d));
</script>
With D3, you get full data-to-graphics mapping. It's better for custom dashboards, data animations, and graphs beyond standard types.
D3 has a learning curve but is industry standard for unique, advanced data visuals. Many chart libraries are built on top of D3.

Chart.js vs D3.js

Chart.jsD3.js
LevelBeginner-friendlyIntermediate/advanced
Charts Bar, line, pie, doughnut, radar, & more (preset types) Anything: bar, line, pie, custom, interactive, animated
Quick Start 2–5 lines & config Write more code for setup
Custom Layouts Limited Full control
Interactivity Good, flexible Great, custom (drag, zoom, etc.)
Mobile Responsive Yes Yes (you code it yourself)
For most dashboards or simple analytics, Chart.js is enough. When you need full creative control or want to build your own chart types, step into D3.

Practice (Show Hidden Solutions)

1. Create a pie chart using Chart.js showing 60 "Passed", 30 "Failed", 10 "Absent".
<!DOCTYPE html> <html lang="en"> <head> <script src="https://cdn.jsdelivr.net/npm/chart.js"></script> </head> <body> <canvas id="myPie" width="310" height="150"></canvas> <script> const ctx = document.getElementById('myPie').getContext('2d'); new Chart(ctx, { type: 'pie', data: { labels: ['Passed', 'Failed', 'Absent'], datasets: [{ data: [60, 30, 10], backgroundColor: ['#38bdf8','#fb7185','#cbd5e1'] }] } }); </script> </body> </html>
2. With D3, draw three bars for the array [4, 9, 5] (heights proportional to their value, SVG 190x100).
<!DOCTYPE html> <html lang="en"> <head> <script src="https://d3js.org/d3.v7.min.js"></script> </head> <body> <svg id="barsvg" width="190" height="100"></svg> <script> const data = [4, 9, 5]; const svg = d3.select("#barsvg"); const x = d3.scaleBand().domain([0,1,2]).range([10,170]).padding(0.25); const y = d3.scaleLinear().domain([0, 10]).range([90, 10]); svg.selectAll("rect") .data(data) .enter().append("rect") .attr("x", (d,i)=>x(i)) .attr("y", d=>y(d)) .attr("width", x.bandwidth()) .attr("height", d=>90-y(d)) .attr("fill", "#38bdf8"); </script> </body> </html>
3. Using Chart.js, create a line chart of days ["Mon","Tue","Wed","Thu","Fri"] and page views [12,31,22,15,10].
<!DOCTYPE html> <html lang="en"> <head> <script src="https://cdn.jsdelivr.net/npm/chart.js"></script> </head> <body> <canvas id="myLine" width="290" height="150"></canvas> <script> const ctx = document.getElementById('myLine').getContext('2d'); new Chart(ctx, { type: "line", data: { labels: ["Mon","Tue","Wed","Thu","Fri"], datasets: [{ label: "Page Views", data: [12,31,22,15,10], backgroundColor: "#bae6fd", borderColor: "#0ea5e9" }] } }); </script> </body> </html>
4. With D3, draw a circle for each item in [12, 22, 44] with the radius proportional to its value.
<!DOCTYPE html> <html lang="en"> <head> <script src="https://d3js.org/d3.v7.min.js"></script> </head> <body> <svg id="circlesvg" width="250" height="70"></svg> <script> const data = [12,22,44]; const svg = d3.select("#circlesvg"); svg.selectAll("circle") .data(data) .enter().append("circle") .attr("cx",(d,i)=>30+70*i) .attr("cy",40) .attr("r",d=>d/2) .attr("fill","#10b981") .attr("opacity",0.7); </script> </body> </html>
Try using empty HTML files and copy the solutions one by one. Demo charts draw best in a web browser—and you can experiment with your own colors and data arrays!
