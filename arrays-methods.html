Putting the Puzzle Together

So far, you've worked with variables, loops, conditionals, and simple data types. What if you need to store a groupâ€”like a list of colors, scores, or students? That's where arrays step in. They help you level up your coding: combining memory, math, and logic into dynamic, real-world apps.

Why Arrays?

An array is like a storage box with slots, each slot holding a value. Want to keep track of all your grades, or every button on a page? Use an array. Think: let fruits = ["apple", "banana", "kiwi"] 

let scores = [98, 87, 79];
console.log(scores[0]); // 98 (first item)
Arrays keep related data together and make it easy to add, remove, or update information in bulk.

Creating Arrays

You can set up an array with square brackets [] and separate items by commas. Items can be numbers, text, or even other arrays.

let fruits = ["apple", "banana", "kiwi"];
let mixed = [8, "cat", true, [1,2]];
Tip: You can have values of any type, but usually keep one kind for clarity.

Reading & Updating Array Items

Array positions start at 0. Access with square brackets, and re-assign directly. 

let pets = ["dog", "cat", "parrot"];
let secondPet = pets[1];      // "cat"
pets[2] = "hamster";          // Change "parrot" to "hamster"
Why? Arrays let you keep, change, and refer to any item by its position. That makes organizing lots of data much simpler.

Looping Over Arrays

Arrays become powerful when you process every value inside them. You can use a for loop, for...of, or forEach to move through each item.

let colors = ["red", "blue", "yellow"];
for (let i = 0; i < colors.length; i++) {
  console.log(colors[i]);
}
// or
for (let color of colors) {
  console.log(color);
}

Common Array Methods

push() adds an item to the end

let fruits = ["apple", "banana"];
fruits.push("lemon");
// ["apple", "banana", "lemon"]
pop() removes the last item

let nums = [10, 20, 30];
let lastNum = nums.pop();
// nums = [10, 20] , lastNum = 30
shift() and unshift(): add/remove from the start

let list = [1, 2, 3];
list.unshift(0); // [0,1,2,3]
list.shift();    // returns and removes 0, so [1,2,3]
indexOf(): find the position of an item

let birds = ["parrot", "canary", "hawk"];
birds.indexOf("canary"); // 1
includes(): check if an item is present

let tools = ["pen", "book"];
tools.includes("book"); // true
join(): turn an array into a string

let letters = ["A", "B", "C"];
let word = letters.join("-"); // "A-B-C"
slice(): copy out a part of an array

let nums = [1, 2, 3, 4];
let part = nums.slice(1, 3); // [2,3]
map(): make a new array by changing each item

let nums = [1, 2, 3];
let doubled = nums.map(function(n) { return n*2; }); // [2,4,6]
filter(): keep only items that pass a test

let nums = [1, 2, 3, 4];
let even = nums.filter(n => n % 2 === 0); // [2,4]
